# Hotel Casa Bella - Sistema de Reservación

Este proyecto implementa un sistema de reservación para hotel usando Python y SQLite (sin dependencias externas).

## Funcionalidades

- Crear reservaciones con validaciones de negocio.
- Listar reservaciones almacenadas en base de datos SQLite.
- Ejecutar en modo consola para uso rápido.

## Ejecutar

```bash
python3 app.py
```

## Ejecutar pruebas

```bash
python3 -m unittest discover -s tests -v
```
